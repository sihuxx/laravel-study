## 01 환경설정
### (1) XAMPP 설치 (PHP + MySQL 해결)

- 윈도우 키 누르고 "시스템 환경 변수 편집" 검색 -> 실행.
- 환경 변수(N) 클릭.
- 시스템 변수(S) 목록에서 Path 찾아서 더블 클릭.
- 새로 만들기(N) 클릭 -> C:\xampp\php 입력 후 확인.
- 컴퓨터 재부팅(또는 CMD 껐다 켜기).

### (2) Composer 설치 (패키지 매니저)

- GetComposer.org 접속 -> Composer-Setup.exe 다운로드 & 설치.
- 설치 중 Choose the command line PHP 단계에서 자동으로 XAMPP의 PHP(C:\xampp\php\php.exe)를 잡기

### (4) 환경 설정 (.env)
#### .env
```bash
DB_CONNECTION=mysql # mysql로 수정
DB_HOST=127.0.0.1
DB_PORT=3306 # 만약 포트번호가 다르면 수정
DB_DATABASE=laravel_2026-01-03
DB_USERNAME=root
DB_PASSWORD= # 공란
```

#### config/app.php
```php
'timezone' => 'Asia/Seoul', // 변경
```

### (5) 초기 세팅
```bash
- **1.** `"composer create-project laravel/laravel '(프로젝트 이름)'`
- **2.** `env 파일 설정`
- **3.** `"php artisan migrate` # DB 테이블 생성
- **3.** `"php artisan serve` # 서버 실행
```
