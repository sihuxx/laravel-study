## 02 라라벨 명령어
### (1) 기본 명령어
`php artisan serve` - 로컬 개발 서버 구동 (사이트 켜기)

`php artisan route:list` - 현재 등록된 모든 주소(URL) 리스트 확인

`php artisan storage:link` - [필수] 업로드한 이미지가 보이도록 심볼릭 링크 연결

`php artisan optimize:clear` - [필수] 캐시 삭제 (설정 변경 적용 안 될 때 사용)

`php artisan tinker` - 라라벨 코드를 터미널에서 즉시 실행 (DB 테스트용)

### (2) 파일 생성 (Make)
`php artisan make:migration CreatePostsTable` - 데이터베이스 테이블 설계도 생성

`php artisan make:model Post` - 모델 파일 생성

`php artisan make:model Post -m` - 모델과 마이그레이션 파일 동시 생성

`php artisan make:controller PostController` - 컨트롤러 파일 생성

`php artisan make:controller PostController --resource` - CRUD(7가지 기능) 틀이 포함된 컨트롤러 생성

### (3) 데이터베이스 (DB)
`php artisan migrate` - 설계도(Migration)대로 DB 테이블 생성

`php artisan migrate:fresh` - DB 초기화(삭제) 후 다시 생성 (데이터 날아감 주의)

`php artisan migrate:rollback` - 방금 실행한 마이그레이션 취소 (되돌리기)
